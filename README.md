# Winter in Data Science (WiDS) 

Welcome to the **Winter in Data Science (WiDS)** repository! This project contains various assignments, datasets, and notebooks focusing on fundamental data science concepts, problem formulation, and equity classification models.

## 📂 Repository Structure

The repository is organized into different assignment folders:

- **Assignment 1:** Fundamentals
  - Explores basic financial metrics.
  - Contains: `Fundamentals.ipynb`, financial metric datasets (`.csv`).

- **Assignment 2:** Problem Formulation
  - Contains: `Problem formulation.pdf`.

- **Assignment 3:** Equity Classification
  - Focuses EDA and Feature Engineering.
  - Contains: `WIDS_Equity_Classification.ipynb`, `raw_dataset.csv`, `engineered_features.csv`.

## 🚀 Getting Started

To run the notebooks in this repository, you will need Python installed along with Jupyter and standard data science libraries (such as `pandas`, `numpy`, and `scikit-learn`).

1. **Clone the repository:**
   ```bash
   git clone <repository_url>
   cd WIDS
   ```

2. **Install dependencies:**
   It's recommended to create a virtual environment first.
   ```bash
   pip install jupyter pandas numpy scikit-learn matplotlib
   ```

3. **Launch Jupyter:**
   ```bash
   jupyter notebook
   ```
   Navigate to the respective assignment folder and open the `.ipynb` files to view or run the code.
